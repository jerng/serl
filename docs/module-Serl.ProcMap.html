Class: ProcMap

Serl.ProcMap()

new ProcMap()

Intended primarily for use at Node.procMap.

A map of a node's local processes (any instances of Proc which were created by calling spawn/n on that node.

Here we use a Map, because we anticipate frequent lookups based on given Pids. We also expect 'frequent additions and removals of key-value pairs'

Each node must track all its previously spawned processes, and so we want an incrementing ProcMap.counter, to ensure that for the lifetime of a node, N, all of N's local processes have a unique ProcIndex.
