(package) procMap

Keys are Pid instances, Values are Proc instances

Properties:
Name Type Description
counter integer

This helps us avoid duplicate process-indices on the local node, when generating a Pid object.

Making this un-writable would be safer, but more expensive as the value would have to BE RE-CONFIGURED for mutation; and we expect this to change frequently.
